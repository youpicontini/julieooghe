Hover Image provides an easy-to-use functionality to change image on hover.

== Description ==

WARNING FOR VERSION 1.4 USERS: PLEASE UPDATE TO THE LATEST VERSION!

Hover Image makes you able to add hover-images to your WordPress Posts and Pages.
Simply use the shortcode [himage] [/himage] and place two images inside.
The first image can be hyperlink, this will be used for the "himage" too.

If you like Hover Image and wanr to use it in your language, feel free to contact me to create translations! Thanks

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `hover-image` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Change settings under Settings --> Hover Image
1. Place `[himage] [/himage]` in your posts or pages and place two images inside of it. The first image can be hyperlink.

== Frequently Asked Questions ==

= How do I change the behavior from the homage? =

Simply go to Settings --> Hover Image.
There you can simply change the design of your transition.
Note that the transition will be invisible if you set the time to 0.

= How do I use milliseconds? =

Set a leading dot to the seconds field.
E.g. .4 or .9
